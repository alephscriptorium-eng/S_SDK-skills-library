# instancias/

Aquí caen **datos** separados y de-identificados: fixtures que prueban
que otro consumidor puede usar el skill sin el histórico real de un mundo.

Reglas:

- El skill lleva el **método** (marco-agnóstico).
- Esta carpeta lleva la **instancia** (dato parametrizado: mundo →
  parámetro genérico).
- No meter datos identificables de un mundo real en el árbol publicable.
- I20 deja la carpeta vacía (solo este README + `.gitkeep`).
