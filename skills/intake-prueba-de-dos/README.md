# intake-prueba-de-dos

Skill de metodo: convertir un intake en un skill materializable con
contrato explicito y ejemplo sintetico.

Este directorio lleva solo el metodo y una fixture minima. Lo que aun no
este definido queda en `reference/CONTRATO.md` como `<pendiente>`, no se
rellena por intuicion.

Recursos:

- `SKILL.md` — metodo del skill
- `reference/CONTRATO.md` — contrato minimo
- `examples/fixture-intake-prueba-de-dos/` — ejemplo sintetico
