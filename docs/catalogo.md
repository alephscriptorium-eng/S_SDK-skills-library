# Catálogo de skills

Todo el método publicado en `@alephscript/skills-scriptorium`. Cada tarjeta
sale del frontmatter real del `SKILL.md` correspondiente; filtrá por
categoría, alterná **solo estables**, o buscá por nombre, tag o descripción.

Cada skill se **activa** con versión fijada — ver
[Consumo](/guide/consumo) y [Activar un skill](/guide/activar).
